# CSCI4448-OOP-Project
How to use:

  -> Make sure you have OpenGL 2.0
  
  -> Combine the Graphics and Project directories into one
  
  -> Go to the directory in the terminal
  
  -> type "make clean;make"
  
  -> type "./final"

Controls:
  
  ->: turn right
  
  <-: turn left
  
  ^: look up
  
  |
  
  v: look down
  
 w: move forward
 
 s: move backward
